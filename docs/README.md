## 📖 文档说明

### 文档分类

- **🚀 部署类**：面向运维人员，包含部署、配置、监控等内容
- **🏗️ 架构类**：面向开发人员，包含系统设计、模块划分等内容
- **📡 协议类**：面向协议开发人员，包含设备通信协议的详细说明
- **🔌 API类**：面向集成开发人员，包含第三方对接接口文档

### 文档维护

所有文档遵循以下原则：

1. ✅ **简洁明了**：避免冗余，保持精简
2. ✅ **及时更新**：代码变更时同步更新文档
3. ✅ **实例丰富**：提供充足的代码示例
4. ✅ **结构清晰**：使用清晰的标题和目录
