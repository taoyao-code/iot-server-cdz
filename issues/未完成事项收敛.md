# 未完成事项收敛（仅"组网设备"协议）
- 优先级定义：P0（打通联调）/ P1（稳态与质量）/ P2（可选）

## P0（必须完成）
- 封装帧编解码：fcfe/fcff + LEN + CMD + SEQ + DIR + GW_ID + BODY + CHK + fcee
- outbound_queue + ACK/重试（≤1次）+ 冷启续发 + (device_id, seq) 去重
- A1 心跳/时间同步；A3 状态上报；A4 状态查询
- C1 控制（按时或按量）→ ACK；C2 结束上报（原因映射）
- E2 参数设置 / E3 参数查询（关键TLV：阈值/时长）
- F1 异常上报 → ACK

## P1（稳态）
- D1/D2 刷卡流程与余额查询
- B1/B2 组网节点列表管理
- 解析容错与限速：非法包/错校验/越界统计与丢弃策略
- 单测覆盖扩充：跨包/乱序/重发回放；快照断言

## P2（可选/延后）
- C3 按功率策略（仅解析记录）
- G1 OTA 下发（仅通路，不含灰度/回滚）
